# Geometos Neue Font Setup

This directory should contain the Geometos Neue font files.

## Required Font Files

Place the following font files in this directory:

- `GeometosNeue-Regular.woff2` (weight: 400)
- `GeometosNeue-Medium.woff2` (weight: 500)
- `GeometosNeue-SemiBold.woff2` (weight: 600)
- `GeometosNeue-Bold.woff2` (weight: 700)

## Font Format

The fonts should be in `.woff2` format for optimal web performance. If you have `.ttf` or `.otf` files, you can convert them using online tools like:

- [CloudConvert](https://cloudconvert.com/ttf-to-woff2)
- [Font Squirrel Webfont Generator](https://www.fontsquirrel.com/tools/webfont-generator)

## Where to Get Geometos Neue

Geometos Neue is a commercial font designed by Deepak Dogra. You can purchase it from:

- [MyFonts](https://www.myfonts.com/collections/geometos-neue-font-deepak-dogra)
- [Creative Fabrica](https://www.creativefabrica.com/product/geometos-neue/)
